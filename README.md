## ♻️ 프로젝트 소개
* 다양한 폐기물의 정보를 확인 할 수 있는 사이트

## 📲 서버
* https://github.com/benecia2/RecycleApp-Server

## 📅 개발기간
  * 23.08.16 ~ 23.08.29

## 🙎🏻‍♂️ 개발 인원 (2)

## ⚙️ 개발 환경
* **Android Studio**
* `Java 11`
* **IDE** : STS 4.19.0
* **Framework** : Springboot (2.7.14)
* **Database** : MySQL (8.0.33)
* **ORM** : JPA

## 📍 주요 기능
### ✔️ 메인화면
* 폐기물 카테고리별 분류
* 폐기물 검색
* 폐기물 업데이트 현황 목록
* 폐기물 상세보기

### ✔️ 게시판
* 더보기 버튼 페이징 처리
* 게시물 검색
* 게시글 등록
* 댓글 작성
* 게시글 수정 및 삭제
